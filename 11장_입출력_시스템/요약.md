## 스트림
- 데이터의 흐름, 혹은 데이터를 전송하는 소프트웨어 모듈
- 스트림의 양 끝에는 프로그램과 장치 연결
	- 보낸 순서대로 데이터 전달
	- 입출력 기본 단위 : 바이트
### 스트림의 종류
- 입력 스트림
	- 입력 장치, 네트워크, 파일로부터 데이터를 프로그램으로 전달하는 스트림
- 출력 스트림
	- 프로그램에서 출력되는 데이터를 출력 장치, 네트워크, 파일로 전달하는 스트림
### 입출력 스트림 버퍼
- 키 입력 스트림의 버퍼
	- 입력장치로부터 입력된 데이터를 프로그램으로 전달하기 전에 일시 저장
	- \<Enter> 키 : 입력의 끝을 의미, 입력된 시점부터 버퍼에서 프로그램이 읽기 시작
- 스크린 출력 스트림의 버퍼
	- 프로그램에서 출력된 데이터를 출력 장치로 보내기 전에 일시 저장
	- 버퍼가 꽉 차거나 강제 출력 명령 시에 출력 장치에 출력
### 입출력 방식
#### 스트림 입출력 방식
- 스트림 버퍼를 이용한 입출력 방식
- 입력된 키는 버퍼에 저장, \<Enter> 키가 입력되면 프로그램이 버퍼에서 읽어가는 방식
- 출력되는 데이터는 일차적으로 스트림 버퍼에 저장, 꽉 차거나, '\n'을 만나거나, 강제 출력 명령의 경우에만 출력
#### 저 수준 입출력 방식
- 버퍼에 쌓이는게 아닌, 키가 입력되는 즉시 프로그램에게 키 값 전달
	- 게임 등 키 입력이 즉각적으로 필요한 곳에 사용
- 프로그램이 출력하는 즉시 출력 장치에 출력
- 컴파이럴마다 다른 라이브러리나 API 지원
- C++ 표준은 스트림 방식만 지원한다.
### 입출력 클래스
- ios
	- 모든 입출력 스트림 클래스들의 기본 클래스, 스트림 입출력에 필요한 공통 함수와 상수, 멤버 변수 선언
- 문자 단위 스트림
	- istream : 입력
		- 객체
			- cin
		- 멤버 함수
			- get()
			- get(char& ch)
			- get(char* s, int n)
			- get(char* s,int n, char delim = '\n')
			- getline(char* s, int n, char delim = '\n')
			- ignore(int n = 1, int delim = EOF)
			- gcount()
	- ostream : 출력
		- 객체
			- cout
			- cerr
			- clog
		- 멤버 함수
			- put(char)
			- write(char* str, int n)
			- flush()
	- iostream : 입출력
- 파일 단위 스트림
	- ifstream : 입력
	- ofstream : 출력
	- fstream : 입출력
## 포맷 입출력
### 포맷 플래그
- 입출력 스트림에서 입출력 형식을 지정하기 위한 플래그
- cout.setf()로 형식을 지정한다. (플래그를 on 한다.)
- cout.unsetf()로 형식을 해제한다. (플래그를 off 한다.)
### 포맷 함수
- cout.width()로 다음으로 출력되는 것의 넓이를 지정
- cout.fill()로 빈칸을 문자로 채우도록 지정
- cout.precision()로 소수점 자릿수 지정(정수를 포함하여 지정)
### 조작자
- C++ 표준 라이브러리에 구현된 조작자
	=> 입출력 포맷 지정 목적
- 개발자 마의 조작자 작성 가능
	=> 다양한 목적
- 매개 변수 없는 조작자와 매개 변수를 가진 조작자로 구분
- 조작자는 항상 << 또는 >> 연산자와 함께 사용됨
- 매개 변수를 가진 조작자를 사용하려면 반드시 #include \<iomanip> 필요
## 삽입 연산자(<<)
- insertion operator, 삽입자라고도 부름
	- << 연산자는 C++의 기본 연산자 : 정수 시프트 연산자
- ostream 클래스에 중복 작성되어 있음
## 추출 연산자(>>)
- extraction operator
	- \>> 연산자는 C++의 기본 연산자 : 정수 시프트 연산자
- ostream 클래스에 중복 작성되어 있음
## 사용자 정의 조작자 함수 원형
- 매개 변수 없는 조작자의 경우
```cpp
istream& manipulatorFunction(istream& ins)
 // 입력 스트림에 사용되는 조작자 원형
ostream& manipulatorFunction(ostream& ins)
 // 출력 스트림에 사용되는 조작자 원형
```
